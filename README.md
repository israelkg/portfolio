## 📄 Descrição
Este projeto é um portfólio pessoal desenvolvido para apresentar meus projetos, habilidades e experiências na área de desenvolvimento web.  
Com uma interface clara e objetiva, o site foi criado com foco em funcionalidade e organização de conteúdo.

## ✨ Funcionalidades
- 🖼️ Seção **Sobre Mim**
- 💼 Lista de **Projetos** com links diretos para os projetos
- 📧 **Formulário de Contato** com validação básica de campos
- 🌐 Navegação simples entre as seções do site

⚠️ **Observação**: O projeto ainda **não é responsivo**.

## ⚙️ Tecnologias Usadas
- **HTML**: Estrutura do site
- **CSS**: Estilização das páginas
- **JavaScript**: Validação de formulário e interatividade
- **Formspree**: Integração para envio de mensagens

## 📌 Pré-requisitos
- Navegador moderno (Chrome, Firefox, etc.)
- Conexão à internet

## 🚀 Como Usar
- Acesse o portfólio online:  
  [https://israelkg.github.io/portfolio/](https://israelkg.github.io/portfolio/)
- Navegue pelas seções para conhecer os projetos e entrar em contato

## 🤝 Contribuição
Este projeto é pessoal, mas estou aberto a feedbacks!  
Sugestões de melhoria são sempre bem-vindas. 😊

## 📦 Instalação
1. Clone o repositório:
    ```bash
    git clone https://github.com/israelkg/portfolio.git
    ```
2. Navegue até o repositório:
    ```bash
    cd portfolio
    ```
3. Abra o arquivo `index.html` em seu navegador
